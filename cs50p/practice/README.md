# CS50P Week 0 Practice

## Concepts Learned:
1. **Print Function**: `print()` with parameters like `end` and `sep`
2. **Variables**: Storing values, naming conventions
3. **Input**: Getting user input with `input()`
4. **Strings**: 
   - Methods: `.strip()`, `.title()`, `.upper()`, `.lower()`
   - f-strings: `f"Hello {name}"`
   - Escape characters: `\"`, `\n`
5. **Functions**: 
   - Built-in: `print()`, `input()`, `len()`
   - Creating: `def function_name():`
   - Parameters and return values

## Key Takeaways:
- Always test edge cases (extra spaces, empty input)
- f-strings are the most readable formatting method
- Chain string methods: `text.strip().title()`
- Comments and pseudocode help plan your code
